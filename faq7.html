The Report API is a service offered by ICON, Coface Business Information offer. It allows you to programmatically place orders for Business reports on companies, Debtor Risk Assessments, Credit Opinions. You also can monitor and retrieve monitoring notifications on companies.

You didn't find the answer to your question ? Feel free to contact our support teams

Access

An API Keys is associated with a user account. To create an API key, log on to the online application, go to your your profile page and generate an API key.

If this feature is not available, then your account has not yet been enabled for API access - contact us.

Don't worry. You can regenerate an new API key. Log on to the online application, go to your your profile page and generate an API key.

You will be warned that the old one will be revoked.
